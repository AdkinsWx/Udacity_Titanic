Possible Issues with the Data¶

After viewing the table below we can see that not all data points are accounted for. For instance on Line [5], Mr. James Moran's age and cabin number are missing (Other passengers even have multiple cabin numbers).

For this missing cabin data I replaced the NaN with a "unknown" string. This could help with labeling and stop those lines of data being dropped from certain functions.

The missing "Age" data can be addressed in a number of ways:

  1. Replacing with 0. a. This would give rid of the NaN. However, when plotting the data it would artificially skew the data towards 0. This would lead to false interpretations.
  2. Replace with the average age. b. Again this would get rid of the NaN in the data. However, this could skew the data towards the median value.
  3. Drop NaN values from the data set. c. Dropping the NaN value rows would work for Age specific statistics but wouldn't be necessary for every factor.

In the end I chose option 3, droping NaN values from the data for certain statistics. This seemed most reasonable because I won't need the Age data for every aspect of the analysis and thus the NaN's wouldn't be an issue. Thus once I needed Age specific data I could drop the NaN rows and continue my analysis.

Below I prescribed some code to remedy the Cabin columns.

Answering the Question (Exploring the Data)¶

1. What factors made people more likely to survive?¶

First lets start with any gender specific factors. I started by plotting the sample population's make-up by gender. To help compare a survivor rate I plotted the survivor's population make-p by gender as well. This will give us a side-by-side visual comparison.

Many stories from the Titanic state a policy of "Women and children first". Meaning that women and children had priority when boarding life boats. With this in mind we could expect to a higher female survivor rate than their male counterparts.

Limitations with the Data¶

One limitation with the data is that this is only a sample of the whole population. Certain parts of the data could be concentrated in this sample that is truly reflected in the total population. One possible example could be that our sample size captured a larger percentage female survivors than the actual population. An undersized sample could lead to false-positives or over estimate the magnitude of an association.

Another limitation is that we do not know the collection methods used to select this particular sample. Was it alphabetical? Random? Or using some sort of bias? This could be troublesome because certain collection methods may not give a suitable sample population ie there may be an underlying bias in the data.

Final statement¶

If one were to expect the best survivabiity rate for any particular passenger the most important factors would be:

  2. Gender
